# Contributing to SS Fusion Multiplayer

Thank you for your interest in contributing to SS Fusion Multiplayer! We welcome contributions from everyone.

## How to Contribute

### Reporting Bugs

If you find a bug, please create a new issue on GitHub describing:
1.  What happened.
2.  What you expected to happen.
3.  Steps to reproduce the issue.

### Suggesting Enhancements

If you have an idea for a new feature or improvement:
1.  Check if it has already been suggested.
2.  Open a new issue describing your idea.

### Pull Requests

1.  **Fork** the repository.
2.  Create a new **branch** for your feature (`git checkout -b feature/amazing-feature`).
3.  **Commit** your changes.
4.  **Push** to your branch.
5.  Open a **Pull Request**.

## Code Style

*   Use standard C# coding conventions.
*   Keep code clean and readable.
*   Document public methods and classes.

## Development Setup

1.  Install .NET Framework 4.0 or higher.
2.  Open the project in Visual Studio or use the provided `build.bat` script.
3.  Ensure all tests pass (if any).

Thank you for helping make this project better!
